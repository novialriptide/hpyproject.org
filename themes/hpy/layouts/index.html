A modern approach for extending Python in C

Developed by members of the Python Software Foundation.

Performance
Much faster on alternative implementations such as PyPy, GraalPy.
Universal Binaries
Extensions built for the HPy Universal ABI can be loaded unmodified on CPython, PyPy, GraalPython, etc.
Debug Mode
Easily identify common problems such as memory leaks, invalid lifetime of objects, invalid usage of APIs.
